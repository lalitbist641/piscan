# Research & Publication Materials

Academic deliverables for the PIScanner project (kept separate from the open-source code repository).

| File | What it is |
|---|---|
| `PIScanner_Paper.docx` | Full IEEE-format research paper (for ICGETEI-2026 / AIAIA track). |
| `ICGETEI2026_Abstract.md` | Conference abstract for first-round submission. |
| `PIScanner_Presentation.pptx` | 15-slide presentation for viva / conference. |
| `THESIS_METHODOLOGY.md` | Methodology write-up (threat model, design, evaluation). |
| `THESIS_RESULTS.md` | Results section with tables. |
| `ANALYSIS_FINDINGS.md` | Detailed analysis of collected results. |
| `PROJECT_OVERVIEW.md` | Plain-language project explainer. |

**Note:** the open-source repo docs (`README.md`, `COMPARISON.md`, `DATASHEET.md`, `RESULTS.md`, `SECURITY.md`, etc.) were intentionally left in the project root because the GitHub README links to them.

## Submission checklist
1. Fill in author names + institution in the paper and abstract.
2. Submit the abstract (AIAIA track): https://forms.gle/369hWDgepmjXKFCE7
3. On acceptance, paste the paper into the official IEEE Word template (two-column) and submit the full paper by the deadline.
4. Register and present using the .pptx.
